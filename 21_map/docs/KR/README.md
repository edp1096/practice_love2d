## 🎯 프로젝트 철학

이 프로젝트는 **"RPG 메이커" 스타일 아키텍처**를 따릅니다:

### **엔진 (재사용 가능)**
`engine/` 폴더는 모든 LÖVE2D 프로젝트에서 재사용 가능한 범용 게임 시스템을 포함합니다:
- 물리 & 충돌 시스템
- 입력 처리 (키보드, 게임패드, 터치)
- 오디오 관리 (BGM, SFX)
- 저장/로드 시스템
- 씬 관리
- UI 렌더링 (HUD, 미니맵, 대화창)

### **게임 (콘텐츠)**
`game/` 폴더는 게임 특화 콘텐츠를 포함합니다:
- 씬 (메뉴, 게임플레이, 설정)
- 엔티티 (플레이어, 적, NPC, 아이템)
- 데이터 (사운드 정의, 입력 설정, 인트로 컷신)

### **장점**
- **새 게임 만들기 쉬움**: `engine/` 폴더 복사하고 새 `game/` 콘텐츠 생성
- **깔끔한 분리**: 엔진 코드 vs 게임 콘텐츠
- **유지보수 용이**: 명확한 구조로 파일 빠르게 찾기
- **RPG 메이커 워크플로우**: 엔진 코드가 아닌 콘텐츠에 집중
